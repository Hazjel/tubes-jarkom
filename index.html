Visualisasi Single Thread vs Multi Thread

Single Thread

Pada single thread, tugas diproses satu per satu secara berurutan. Tugas berikutnya harus menunggu tugas sebelumnya selesai.

Multi Thread

Pada multi thread, beberapa tugas dapat diproses secara bersamaan. Tidak perlu menunggu tugas lain selesai untuk memulai tugas baru.
